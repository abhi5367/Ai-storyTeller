# ⚡ SwiftScribe AI: The Master Story Teller

SwiftScribe AI is a high-performance storytelling application that generates cinematic, children-friendly stories using **Llama 3.3-70B** via the Groq Cloud API. It features a custom-styled, dark-themed UI with a focus on immersive storytelling.

## 🚀 Features
- **Cinematic UI:** Fixed dark theme with a glassmorphism overlay and a storytelling-inspired background.
- **Llama 3.3 Integration:** Powered by the Groq SDK for near-instant text generation.
- **Strict Formatting:** Uses Regex parsing to ensure the AI always delivers a clean Title and Story body without markdown noise.
- **Responsive Container:** Scrollable story cards designed for long-form narrative reading.
